# Notebooks

## training_brainnet.ipynb
The main Google Colab notebook used for:
- Dataset download from Kaggle
- EfficientNetB0 training (86.4% accuracy)
- ResNet50V2 training (94.6% accuracy)  
- SCC analysis on 1,600 test images
- GradCAM++ visualization
- Results and plots

**Note**: Trained model files (.keras) are stored in Google Drive.
They are not included in this repository due to file size limits.
Models can be re-trained using the notebook.
